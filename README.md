# mObywatel Scraper

## Setup
- **1)** `chmod +x setup.sh`
- **2)** `sudo ./setup.sh`

## Usage
- **1)** `python3 main.py`
- **2)** browser window with login form to *mObywatel* should open
- **3)** chose your login method and proceed
- **4)** after succesful login, program will gather data and print result to console
- **5)** profit
